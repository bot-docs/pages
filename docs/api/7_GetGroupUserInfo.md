# 获取群组用户昵称和头像

## 基本信息

**Path：** /api/v1/GetGroupUserInfo

**Method：** POST

**接口描述：**

获取群组用户昵称和头像。

## 请求参数

**Headers**

| 参数名称          | 参数值              | 是否必须 | 示例 | 备注 |
|---------------|------------------|------|----|----|
| Content-Type  | application/json | 是    |    |    |
| Authorization | token            | 是    |    |    |

**Body**

| 名称           | 类型        | 是否必须 | 默认值 | 备注                             | 其他信息                       |
|--------------|-----------|------|-----|--------------------------------|----------------------------|
| gid           | integer   | 必须   |     | 群组ID                            |                            |
| uid        | integer    | 必须   |     | 用户ID                          |                            |



## 返回数据

| 名称  | 类型      | 是否必须 | 默认值 | 备注                   | 其他信息 |
|-----|---------|------|-----|----------------------|------|
| ret | integer | 必须   | 0   |                      |      |
| msg | string  | 必须   | ok  |                      |      |
| info    | object  | 非必须  |     |        | {}        |
| ├─ uid        | string  | 非必须  |     | 用户ID   |           |
| ├─ avatar     | string  | 非必须  |     | 用户头像   |           |
| ├─ nickname       | string  | 非必须  |     | 用户名称   |           |
